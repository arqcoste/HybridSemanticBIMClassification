# Hybrid Semantic BIM Classification Engine

A hybrid BIM classification engine combining:

- Domain semantic embeddings
- IFC semantic structure
- Rule-based reasoning

The system aims to classify BIM elements into Uniclass 2015 tables using a hybrid approach.

## Research Goals

1. Improve BIM element classification accuracy
2. Combine deterministic rules with semantic embeddings
3. Leverage IFC semantics as contextual signals

## Structure

- semantic → embedding-based classification
- rules → deterministic rule engine
- hybrid → decision fusion layer
